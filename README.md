# FSSCREENER — Full Stride 馬市場検索ツール

Full Stride の馬市場を検索するファンメイドツールです。
予算・脚質・距離・芝ダート・毛色・成長タイプ・8軸ステータスから候補馬を絞り込み、気になった馬をタップすると現実の競走馬としての戦績・エピソードが見られます。

スマホでの利用を推奨します。

## 使い方

1. 予算上限を入力（必須）
2. 脚質・距離・芝ダート・毛色・成長タイプ・8軸ステータスは、気になる項目だけ任意で絞り込み
3. 「検索する」を押すと候補馬が一覧で出てきます
4. 気になる馬をタップすると、現実の競走馬としての戦績やエピソードが見られます

日本語/英語切替、ライト/ダークテーマにも対応しています（史実紹介コラムは日本語のみ）。

本ツールはファンによる非公式検索ツールです。
